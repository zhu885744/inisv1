## QPS告警管理 API

### 接口概述

QPS告警管理控制器，提供QPS告警记录的 CRUD 操作及数据统计功能。

### 状态码规范

| 状态码 | 含义 | 使用场景 |
| :--- | :--- | :--- |
| 200 | 请求成功 | 成功响应数据 |
| 201 | 创建成功 | 资源创建成功 |
| 202 | 接受请求 | 无实际功能接口 |
| 204 | 无数据 | 查询无结果或无可操作数据 |
| 400 | 请求参数错误 | 参数校验失败、数据操作失败 |
| 405 | 方法不允许 | 调用了不存在的方法 |
| 500 | 服务器错误 | 服务器内部错误 |
